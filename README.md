## MERN STACK TODO APP

## Features

- Add a new ToDo
- Update ToDo
- Delete ToDo
- Get all the ToDo


## Tech-Stack used

- MongoDB
- ExpressJS
- React
- NodejS

## start the project (using docker-compose) 
- add this link to the .env file in the backend :
 MONGODB_URI="mongodb://mongo:27017/mern-app"
- docker-compose up --build
